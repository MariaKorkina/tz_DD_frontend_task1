Gulp — это таск-менеджер для автоматического выполнения часто используемых задач, написанный на языке программирования JavaScript.

Обзор технологии

Gulp — это программа для выполнения задач, которая использует Node.js в качестве платформы. Gulp использует только код JavaScript и помогает запускать внешние задачи и крупномасштабные веб-приложения. Он создает системные автоматизированные задачи, такие как минимизация CSS и HTML, объединение библиотечных файлов и компиляция файлов SASS. Эти задачи можно запустить с помощью сценариев Shell или Bash в командной строке.

История

  • Gulp v1.0.0 был выпущен 15 января 2015 года
  • Gulp.js 3.9.1 был единственной версией много лет. И хотя был доступен Gulp 4, его нужно было устанавливать с помощью npm install gulp@next
  • 10 декабря 2018 года Gulp.js 4.0 был объявлен единственной версией и появился в менеджере пакетов npm

Характеристики

  • Обеспечивает минификацию и конкатенацию
  • Использует чистый код JavaScrip
  • Преобразует LESS или SASS в компиляцию CSS
  • Управляет манипуляциями с файлами в памяти и повышает скорость с помощью платформы Node.js
  • Оптимизация ресурсов таких как javascript, css, изображения
  • Автоматическая перезагрузка браузера при сохранении файла
  • Создание карт проекта
  • Использование автопрефиксера

В него можно писать любой javascript код, но gulp — это менеджер управления задачами, и эти задачи описываются в этом файле в виде обычных функций. Преимущества

  • Быстродействие
  • Легок в понимании и кодировании
  • Легко тестировать веб-приложения
  • Плагины просты в использовании, предназначены для одновременного выполнения одного действия
  • Многократное выполнение повторяющихся задач

Недостатки

  • Большое количество зависимостей
  • Невозможность выполнения нескольких задач
  • Не такая чистая конфигурация, как у Grant
